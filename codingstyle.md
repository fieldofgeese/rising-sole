# Coding Style

**Indent:** Tab vs spaces? Hur många spaces för indent, eller hur många spaces per tab?

Namn av variabler:
```c
int min_variabel = 0;
int minVariabel = 0;
int MinVariabel = 0;
```

Namn av funktioner
```c
min_funktion();
minFunktion();
MinFunktion();
```

Namn av structs
```c
struct min_struct ...
MinStruct ...
```

Braces
```c
for (...) {
}

for (...)
{
}
```

Pointers
```c
int* ptr;
int *ptr;
```
